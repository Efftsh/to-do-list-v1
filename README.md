<h2>📌 Project Description</h2>
<p>
<strong>To-Do List v1</strong> is a simple task management web application built with
HTML, CSS, and JavaScript. Users can add tasks, mark them as completed,
and delete them from the list. The application dynamically updates the task
list using JavaScript without refreshing the page.
</p>

<h2>🛠 Technologies</h2>
<ul>
  <li>HTML5 – Structure of the application</li>
  <li>CSS3 – Layout and styling</li>
  <li>JavaScript (Vanilla JS) – Application logic</li>
  <li>DOM Manipulation – Dynamically updating elements</li>
</ul>

<h2>✨ Features</h2>
<ul>
  <li>Add new tasks to the list</li>
  <li>Delete tasks from the list</li>
  <li>Mark tasks as completed</li>
  <li>Dynamic DOM updates without refreshing the page</li>
  <li>Clean and simple two-column layout</li>
  <li>Visual indicator for completed tasks</li>
</ul>

<h2>⚙️ The Process</h2>
<ol>
  <li>Created the layout of the application using HTML.</li>
  <li>Styled the interface using CSS with a simple grid layout.</li>
  <li>Implemented functionality using JavaScript.</li>
  <li>Used DOM manipulation to dynamically create and manage task elements.</li>
  <li>Added event listeners for adding, deleting, and marking tasks as completed.</li>
</ol>
